
# adult.json
针对OnePlayer的成人资源。



# 使用方法
### 本地导入
1. 下载adult.json
2. 【设置】-【本地导入影视源】-【选择下载好的adult.json】



### 网络导入
1. 【设置】-【网络导入影视源】
2. 输入下面网址<br>
https://raw.githubusercontent.com/gnrevival/videoSource/main/adult.json
